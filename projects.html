NLP – Humor Detection

This Natural Language Processing project focused on detecting humor in movie reviews using a large subset of the IMDb dataset. I implemented transformer-based models for classification, fine-tuning them for humor detection in text. Given the limited GPU resources, I explored zero-shot and few-shot learning techniques to maximize model performance without extensive training. This allowed me to evaluate how transformer models like BART or RoBERTa perform when generalizing to niche tasks. I preprocessed the data for tokenization and cleaning, and implemented evaluation metrics such as F1-score and accuracy. The project challenged me to work with real-world constraints and taught me how to manage large-scale NLP tasks efficiently, optimizing for both resource usage and accuracy.

TFIDF

In this project, I built a Java-based search system for an online library using the TF-IDF (Term Frequency–Inverse Document Frequency) algorithm to retrieve and rank relevant documents. The system included two interfaces—Text and Library—used to compare performance and analyze how different data structures affect search efficiency. I experimented with structures such as TreeMap, HashMap, and ArrayList to evaluate their impact on both runtime and memory usage across various tasks like indexing, term frequency counting, and ranking. The focus was on determining which data structure worked best under each interface in terms of time and space complexity.
